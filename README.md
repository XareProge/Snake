# Змейка

Классическая игра «Змейка» на C++ с графикой на SFML.

## Управление

| Клавиша | Действие |
|---|---|
| `W` / `↑` | Вверх |
| `S` / `↓` | Вниз |
| `A` / `←` | Влево |
| `D` / `→` | Вправо |
| `Enter` | Подтвердить / Заново |
| `Esc` | Меню / Назад |
| `R` | Таблица рекордов |

## Уровни сложности

| Уровень | Препятствия | Особые объекты |
|---|---|---|
| Безобидный | — | — |
| Лёгкий | Стены | — |
| Средний | Стены | Ядовитая еда (−15 очков) |
| Сложный | Стены | Яд + Телепорт + Инверсия управления |

## Особые объекты

- 🔴 **Яблоко** — +10 очков, змейка растёт
- 🟣 **Яд** — −15 очков, змейка укорачивается
- 🔵 **Телепорт** — змейка перемещается в случайное место
- 🟡 **Инверсия** — управление переворачивается на 30 ходов

## Рекорды

Сохраняются в файл `records.txt` рядом с `.exe` в формате:
```
дата|сложность|счёт
```

## Сборка

### Требования
- Visual Studio 2022
- [SFML 2.6.x](https://www.sfml-dev.org/download.php) для Visual C++ 17 — 64-bit

### Установка SFML
Распакуй скачанный архив в папку проекта:
```
Snake/
└── SFML/
    ├── include/
    └── lib/
```
Скопируй `.dll` файлы из `SFML/bin/` в `x64/Debug/`.

### Запуск
Открой `Snake.sln` → конфигурация **Debug | x64** → **F5**

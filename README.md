# Змейка

Классическая игра «Змейка» на C++ с графикой на SFML.

## Управление

| Клавиша | Действие |
|---|---|
| `W` / `↑` | Вверх |
| `S` / `↓` | Вниз |
| `A` / `←` | Влево |
| `D` / `→` | Вправо |
| `Enter` | Подтвердить / Заново |
| `Esc` | Меню / Выход |

## Правила

- Змейка растёт, поедая яблоки (+10 очков за каждое)
- Столкновение со стеной или собой — конец игры
- Скорость постепенно увеличивается по мере роста счёта

## Сборка

### Требования
- Visual Studio 2022
- [SFML 2.6.x](https://www.sfml-dev.org/download.php) для Visual C++ 17 — 64-bit

### Установка SFML
Распакуй скачанный архив SFML в папку проекта:
```
Snake/
└── SFML/
    ├── include/
    └── lib/
```

Скопируй `.dll` файлы из `SFML/bin/` в папку с `.exe` (`x64/Debug/`).

### Запуск
Открой `Snake.sln` → конфигурация **Debug | x64** → **F5**
